Requirements:
1. Install UIPath from https://uipath.com
2. Install the chrome extension for UIPath
3. Check that u have UIPath Studio, UIPath Assistant, and UIPath Diagnostic Tool after installing

Steps to download this bot:
1. Open command prompt
2. Navigate to where u want to store this bot using "cd" commmand
    - for example if i wanted to navigate to the "code" folder i would type **cd "C:/code"**
3. type **git clone https://github.com/e75w/tixbot.git**
4. check that u have a folder called tixbot in that folder

Steps to Use:
0. Go to browser n go to https://www.maiseat.com/en-US/shows/14736 
1. Open UIPath Studio
2. Click Open a local project
3. Open tixbot folder
4. Select project.json
5. Click Open

6. Open Main.xaml
7. Click the down arrow for the sequence labeled Typing Card Details
8. fill in ur card details but make sure u close them in double quotes
   - for example if card number is 1234567890123456 type "1234567890123456"
9. click the triangle to run, make sure it is set to Run File
  
10. hope that it works XD
